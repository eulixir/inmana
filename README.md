## Install Inmana

    --Open terminal--

    #Clone repo from github
    git clone https://github.com/joaopealves/Inmana.git

    #Entry in Rocketpay folder
    cd Inmana

    #Install deps
    mix deps.get

    #if your postgresql is installed correctly, type
    #Create Database and migrations
    mix ecto.setup

    #Open interactive terminal
    iex -S mix

    #Open Phoenix server
    mix phx.server

    #Exec test
    mix test
